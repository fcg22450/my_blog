Python Pandas

Pandas的数据结构

一、Series

1.Series时一种类似于一维数组的对象,主要由以下两个部分 组成:

  • values:一组数据
  • index: 相关的数据索引标签

2.Series创建方式

  1. 由列表

    使用列表创建

    • ```python

      创建一个对象,该对象不指定索引,将会默认使用0—(n-1)的形式

      import pandas as pd
      from pandas import Series,DataFrame
      a_ser = Series(data=[1,2,3,4])
      print(a_ser)
      print(a_ser[‘a’])
  2. 使用numpy数组创建

    使用数组创建

    • ```python

      隐式索引

      import pandas as pd
      from pandas import Series,DataFrame
      import numpy as np
      b_ser = Series(data=np.random.randint(0,100,size=(5,)))
      print(b_ser)

二、切片

显示索引切片与隐式索引切片

  • ```python
    print(a_ser[0:3])

  1. 使用head和tail获取前n个数或者后n个数

    • ```python
      print(b_ser.head(2))

  2. 对Series进行去重

    • ```python
      print(b_ser.unique())

  3. 关于是否为空的判定

    • ```python
      print(s.isnull())
      print(s.notnull())
